### 🔍 Sobre Mí

Soy un **Desarrollador Full Stack** apasionado por la creación de soluciones eficientes y modernas. Me enfoco en arquitecturas **MVC**, desarrollo de herramientas de escritorio con **Python** y aplicaciones web interactivas. 

Actualmente, estoy profundizando en el desarrollo asistido por **IA**, utilizando herramientas avanzadas como **Antigravity** y **Codex** para llevar el código al siguiente nivel. ¡Mi objetivo es fusionar la lógica robusta del backend con interfaces futuristas y funcionales!

- 🌍 Ubicación: **Perú**
- ⚡ Intereses: IA, Automatización, Diseño Tech/Oscuro.
- 🛠️ Enfoque: Código limpio, arquitectura escalable y experiencia de usuario premium.
